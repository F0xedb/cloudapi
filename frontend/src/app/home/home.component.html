Cloud API search!

This project implements the google search api. This API filters searches to social media only. It has different features like searching, pagination, login system and search history.

Google search API

Searching

This search API filters to a default set of websites mainly. youtube, reddit, facebook, twitter, 9gag, instagram, imgur, pintrest. If you wish to change this then you need to rebuild the frontend. Edit the environments file and update CX and API_KEY to the correct values.

View details »

History

Every search you make will be saved inside a database under your user credentials. The history can be retrieved by going to the history tab. There you can search your history, delete entries and much more.
